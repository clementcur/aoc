* [AoC 2024](https://adventofcode.com/2024)

# Setup

- Install python 3.11
- Create a venv to isolate dependancies
```shell script
python3 -m venv .venv
```
- Enable the venv
```shell script
. .venv/bin/activate
```
- Install pip dependancies
```shell script
python -m  pip install -r requirements.txt
python -m  pip install -r requirements_dev.txt
```

# Dev

- Enable the venv
```shell script
. .venv/bin/activate
```
- Start it
```shell script
python main.py
```
